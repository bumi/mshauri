## Code style + linting

### eslint

```bash
node_modules/.bin/eslint --ext .js,.vue app/javascript/ --fix # auto fix
```

### Rubocop

The project uses [rubocop](https://github.com/rubocop-hq/rubocop) for code linting.

```rb

bundle exec rubocop
bundle exec rubocop -a # auto fix
```
